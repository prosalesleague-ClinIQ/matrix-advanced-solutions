# Build Solutions Page

## Structure
- Page hero with "Solutions" eyebrow
- 7 solution sections alternating background

## Solution sections
Each section includes:
1. Icon + badge
2. Section headline (solution subtitle)
3. Description paragraph
4. "Who It Serves" block
5. "Why Clinics Add This" block
6. "How Matrix Supports" block
7. Categories grid (2-col)
8. CTA button
9. Compliance note

## Categories (from data/solutions.ts)
- Peptides, Exosomes, Metabolic/GLP, Sexual Health, Regenerative, Longevity, Clinic Growth

## Copy rules
- B2B clinic-facing language
- No consumer supplement tone
- No risky medical claims
- Compliance note per section

## Layout rules
- Alternate bg-navy-900/30 on odd sections
- 2-column grid: content left, categories right
- Strong visual hierarchy per section
- Final CTA section at bottom
