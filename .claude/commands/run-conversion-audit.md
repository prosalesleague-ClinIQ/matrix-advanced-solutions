# Run Conversion Audit

Review the site for conversion effectiveness:

## Clarity
- [ ] Is the homepage value proposition immediately clear?
- [ ] Can a visitor understand what Matrix does within 5 seconds?
- [ ] Are solution categories easy to scan?

## CTA prominence
- [ ] Are primary CTAs visible on every major section?
- [ ] Is the CTA hierarchy consistent (catalog > strategy > onboarding)?
- [ ] Do CTAs use clear, action-oriented language?

## Trust signals
- [ ] Compliance language present on conversion surfaces?
- [ ] "Professional use only" visible where needed?
- [ ] Trust section communicates standards clearly?

## Form friction
- [ ] Are forms the right length for their intent?
- [ ] Request Catalog feels fast and low-friction?
- [ ] Onboarding feels detailed but not overwhelming?
- [ ] Success messages communicate clear next steps?

## Funnel continuity
- [ ] Homepage → Solutions → Catalog → Contact flow is smooth?
- [ ] Growth Infrastructure → Onboarding path is clear?
- [ ] Peptide Map → Catalog inquiry path exists?

## Value communication
- [ ] "More than supply" message lands on Growth page?
- [ ] Solutions page shows breadth without overwhelm?
- [ ] About page communicates strategic philosophy?

## Enterprise positioning
- [ ] Does the site feel enterprise-grade, not startup-y?
- [ ] Is the B2B framing consistent across all pages?
- [ ] Would a clinic owner feel this is a serious partner?
