# Matrix Design System Skill

## Visual identity
- Black and near-black surfaces, deep navy foundations, steel gray structure
- Restrained cool-blue (#3b82f6) and cyan (#22d3ee) accents
- Soft premium gradients with restraint, layered shadows
- Glass only when it improves depth and hierarchy

## Typography
- Hero headline: bold, 4xl–7xl depending on breakpoint
- Section headline: 2xl–5xl
- Section subhead: base–xl
- Eyebrow: uppercase, tracking-[0.2em], text-xs, text-accent-blue
- Body: base–lg, text-steel-400
- Fine print: sm, text-steel-600

## Spacing
- Section padding: py-16 to py-20 mobile, py-24 to py-32 desktop
- Container: max-w-7xl content, max-w-4xl narrative, max-w-3xl long text
- Clear separation between eyebrow → headline → subhead → content → CTA

## Grid
- 2–3 column card systems, 4-column for compact items
- Asymmetric hero layouts where one side carries visual mass

## Card design
- rounded-2xl, subtle border (border-white/8), hover lift (-translate-y-1)
- bg-surface-card, hover:bg-surface-elevated, hover:border-white/15
- Optional glow: hover:shadow-[0_0_30px_rgba(59,130,246,0.12)]

## CTA hierarchy
- Primary: filled bg-accent-blue, white text, shadow
- Secondary: ghost/outline with border-white/20
- Tertiary: text link with icon

## Hover behavior
- Subtle y-translation, border brightening, shadow change
- No large rotations, aggressive scaling, or fast flashy motion

## Prohibited styles
- Rainbow gradients, loud neon, crypto glow, clip-art medical visuals
- Generic startup illustrations, white-card-on-white layouts
- Flat lifeless cards, oversized backdrop blur

## Peptide Map & MuscleLock alignment
- Inherit shared nav, footer, typography, spacing, button system
- Preserve Peptide Map as education destination
- Preserve MuscleLock as product-storytelling destination
- Do not flatten them into generic pages

## Section composition pattern
1. eyebrow → 2. headline → 3. subhead → 4. content grid → 5. CTA/proof
